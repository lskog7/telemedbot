Список комманд для бота.

1. /help - описание всех функций.
2. /botinfo - вывод информации о том, что делает бот.
3. /results - вывод списка всех результатов с датами для данного пользователя.
4. /test - начать тестирование.
5. /myanswers - выводит список тестирований для пользователя. Выбирает тест и видит свои ответы.
6. /userinfo - выводит информацию о зарегистрированном пользователе. (с кнопкой "изменить данные")
